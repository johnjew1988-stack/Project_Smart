## Project Structure
src/gateway - Core gateway logic
src/api - REST API endpoints
src/devices - Device drivers
src/utils - Utility functions
tests/ - Test suites
docs/ - Documentation
config/ - Configuration files

## Getting Started
Prerequisites: Node.js v14+, npm/yarn, Git

## Installation
1. Clone: git clone https://github.com/johnjew1988-stack/Project_Smart.git
2. Install: npm install
3. Configure: cp .env.example .env
4. Initialize: npm run db:init

## Usage
Start: npm start
Development: npm run dev
Tests: npm test

For more details see the docs/README.md file.
